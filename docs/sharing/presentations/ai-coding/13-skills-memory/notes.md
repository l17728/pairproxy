两个机制，一个管"知道什么"，一个管"记住什么"。

Skills 的核心思路是**渐进式知识披露**。别把所有规则一股脑塞进 AGENTS.md——context 有上限的。按需加载就好：碰到 PDF 任务加载 pdf skill，碰到邮件任务加载 email skill。

记忆闭环是我觉得整个方法论里**最有复利效应**的部分。流程是：踩坑 → 复盘归因 → 举一反三普查同类 → 写进 AGENTS.md 规则化 → 同类 bug 永不复发。

每个 bug 都是一次加强 Harness 的机会。**浪费 bug 就是浪费最贵的学习。**

我举一反三的方式不是随便看看——是搜相同模式，逐一确认，每个都补测试。这样下次 AI 在同一个地方就不会再犯了。
