先说我们做了什么，再说怎么做的。

我们团队用 Claude Code，十几个人共享一个 Anthropic API Key。很快我就发现一个问题——谁用了多少 token，有没有超额，费用怎么分摊，完全不知道。

我就想，能不能做一个透明代理，架在 Claude Code 和 Anthropic 之间，把这些事全管起来。

这就是 PairProxy。架构很简单：一个 cproxy 跑在每个人本机，一个 sproxy 跑在服务器上。用户设两个环境变量，其他完全无感知。

这里有一个我觉得很重要的原则：**你练手的项目，一定要解决你自己的真实问题。** 因为只有你自己才知道什么叫"够好"。demo 谁都能糊弄过去，生产系统不行。
